🌐 محرر HTML مباشر - موقع متقدم

نظرة عامة

محرر HTML مباشر متطور يتيح لك كتابة وتجربة كود HTML بشكل فوري مع معاينة حية، ووحدة تحكم (Console) متكاملة، وعرض ثلاثي الأبعاد للصفحات. تم تصميم هذا المحرر لتجربة تطوير ويب سلسة وفعالة.

✨ المميزات الرئيسية

📝 محرر متقدم

· محرر Ace Editor - محرر نصوص احترافي مع تظليل بناء الجملة
· دعم HTML كامل - كتابة وتحرير كود HTML بسهولة
· خطوط عربية مدعومة - دعم كامل للغة العربية
· ثيمات متعددة (Twilight, Monokai, Textmate, Chrome)

🖥️ معاينة فورية

· تشغيل مباشر - عرض النتائج فوراً (Ctrl+Enter)
· وضع التشغيل الآلي - تحديث تلقائي مع كل تغيير
· عرض ثلاثي الأبعاد (3D) - تجربة عرض مميزة
· نافذة منبثقة - عرض منفصل مع تأثيرات 3D

🛠️ أدوات المطور

· كونسول مدمج - عرض رسائل console.log() والأخطاء
· عرض الأخطاء - التقاط وعرض أخطاء JavaScript
· مسح الكونسول - تنظيف سريع للرسائل

💾 إدارة الملفات

· فتح ملفات HTML - تحميل ملفات محلية
· حفظ الملفات - حفظ الكود كملف HTML
· إنشاء ملف جديد - بدء مشروع جديد

🎨 واجهة مستخدم أنيقة

· تصميم داكن احترافي - مريح للعين
· واجهة مجاوبة - تعمل على جميع الأجهزة
· تأثيرات بصرية - ظلال ونعومة في التصميم

🎯 كيفية الاستخدام

البدء السريع

1. افتح الموقع في المتصفح
2. اكتب كود HTML في المحرر الأيمن
3. اضغط على "تشغيل" أو Ctrl+Enter لمشاهدة النتيجة
4. استخدم المعاينة لمشاهدة النتائج فورياً

الميزات المتقدمة

🖱️ التشغيل الآلي

· فعّل زر "تشغيل آلي" لتحديث المعاينة تلقائياً
· مثالي للتطوير السريع والتجارب

🎯 عرض ثلاثي الأبعاد

· اختر "صفحة داخلية (3D)" أو "نافذة منبثقة"
· استمتع بعرض احترافي مع تأثيرات ثلاثية الأبعاد

📱 وضع الأجهزة المحمولة

· الموقع متجاوب بالكامل
· يعمل بشكل مثالي على الهواتف والأجهزة اللوحية

⌨️ اختصارات لوحة المفاتيح

الاختصار الوظيفة
Ctrl+Enter تشغيل الكود
Ctrl+S حفظ الملف
Ctrl+O فتح ملف
Ctrl+N مشروع جديد

🔧 حل المشكلات الشائعة

❌ الكود لا يعمل في المعاينة

· تأكد من كتابة كود HTML صحيح
· تحقق من وجود علامات <html> الأساسية
· استخدم "تشغيل" يدوياً (Ctrl+Enter)

❌ console.log() لا يظهر

· تأكد من فتح الكونسول في الأسفل
· استخدم مسح الكونسول ثم أعد التشغيل
· تحقق من إعدادات الأمان في المتصفح

❌ الأخطاء لا تظهر

· قد تكون محجوبة بواسطة سياسة CORS
· استخدم وضع "تطوير" في المتصفح
· تأكد من تفعيل sandbox="allow-scripts"
